The Protein Feature View

SVG library for visualising protein annotations

This is the library behind the Protein Feature View at RCSB PDB.

Dependencies

* jquery (V. 1.7.2)
* jqueryui (V. 1.8.22)
* jquery SVG (V. 1.4.5)
